# Mod_AgeOfWonders4_ReaverFixes
I hope putting game mods I've made onto Git isn't weird, but I need a place to store the files safely and track the changes made to them.

[Check out the live Steam Workshop mod here](https://steamcommunity.com/sharedfiles/filedetails/?id=3684419609)

## Overview
Age of Wonders 4 is a popular strategy game, wherein one of the factions, the Reavers, fell short on gameplay even though their design concept is incredible. Using the game's modding tools, I made a series of fixes taking into account long-standing community gripes and feedback, to create a smoother, more enjoyable experience for the players.

## Community Response
* It made the 'Most Popular' list of mods in under a week, and is currently sitting at #4
* A large number of community discussions and debate has been created and listened to
* We're going to keep trying to find the sweet spot and improve player experience :)
